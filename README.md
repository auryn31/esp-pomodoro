
# ESP32 Pomodoro timer

## Needed Hardware

- ESP32 C3 Super Mini 3€
- On/Off Switch 40ct
- Screws + Insert 30ct
- TZT OLES IIC 0.96" 2€
- Loading board 5V Lithium Battery 1.8€
- Tactile Push Button 10ct
- 5V step up
